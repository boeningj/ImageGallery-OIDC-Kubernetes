# ImageGallery.Infrastructure

Shared infrastructure components for the ImageGallery platform.

## Responsibilities

- ASP.NET Core Data Protection key persistence
- Shared security infrastructure
- Future signing key persistence
- Infrastructure-specific EF Core migrations

## Distributed Authentication

ASP.NET Core Data Protection keys are persisted to shared SQL Server storage to support distributed authentication scenarios.

This allows multiple application instances/pods to share the same cryptographic key ring, enabling:

- Durable authentication cookies
- Restart-safe authentication
- Kubernetes pod replacement
- Rolling deployments
- Horizontal Pod Autoscaling (HPA)
- Multi-instance authentication compatibility

Without shared persistence, authentication cookies generated by one application instance could not be decrypted by another instance.

## EF Core Migration Commands

Add migration:

```powershell
dotnet ef migrations add InitialDataProtectionKeys --project .\ImageGallery.Infrastructure\ --startup-project .\ImageGallery.Client\
```

Apply migration:

```powershell
dotnet ef database update --project .\ImageGallery.Infrastructure\ --startup-project .\ImageGallery.Client\
```
